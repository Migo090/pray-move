# Pray & Move 🕌🏃‍♂️

**Pray & Move** is a holistic productivity companion designed by **Magdy Ali**. It seamlessly integrates spiritual mindfulness with physical health by providing accurate prayer timings and movement reminders directly in your browser.

[![GitHub License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)
[![Platform](https://img.shields.io/badge/platform-Firefox%20%7C%20Chrome-orange.svg)](#)

## ✨ Features
* **Real-Time Prayer Dashboard:** Automatic location detection ensures you stay in sync with your local Masjid.
* **12-Hour Time Format:** A clean, modern interface showing Fajr through Isha in user-friendly 12-hour time.
* **Movement "Flow" Reminders:** Customizable intervals (default 30 mins) to encourage stretching and eye-rest during long work sessions.
* **Custom Toast Notifications:** High-fidelity, non-intrusive HTML/CSS notifications that slide into view and stay until acknowledged.
* **Auditory Alerts:** A gentle bell sound to notify you when it's time for prayer or a movement break.

## 🛠 Installation

### Firefox
1. Download the latest `.xpi` file from the [Releases](https://github.com/Migo090/pray-move/releases) section.
2. Open Firefox and navigate to `about:addons`.
3. Click the Gear icon (⚙️) and select **"Install Add-on From File..."**.

### Chrome
1. Download this repository as a ZIP and extract it.
2. Navigate to `chrome://extensions`.
3. Enable **"Developer mode"** in the top right.
4. Click **"Load unpacked"** and select the extracted folder.

## 🔒 Privacy & Data Handling
**Pray & Move** is built with a "Privacy First" philosophy:
* **Zero Data Collection:** No user data is tracked, collected, or sold.
* **Local Processing:** Your city name is used only to fetch timings from the Aladhan API. No unique identifiers are sent.
* **Local Storage:** All preferences and cached timings are stored strictly on your device using `browser.storage.local`.
* **Open Source:** The code is fully transparent and available for audit here on GitHub.
